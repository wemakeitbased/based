# Based landing page

A Vercel-ready Next.js landing page inspired by the structure of glowdaily.app, but rewritten and redesigned for **Based**.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

### Option 1
1. Create a new GitHub repo
2. Upload these files
3. Import the repo into Vercel
4. Click Deploy

### Option 2
1. Install Vercel CLI
2. Run:

```bash
npm install
npx vercel
```

## Recommended customizations

- Replace the placeholder phone mockup with real Based app screenshots
- Update metadata in `app/layout.tsx`
- Swap the CTA link for your real App Store URL
- Add privacy policy and terms pages
